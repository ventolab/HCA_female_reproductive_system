# Immune | HFRSCA v1

This directory contains the scripts used for integrating, annontating, and analysing the immune cells in the Human Female Reproductive System Cell Atlas v1 (across lifespan, excluding donors taking exogenous hormones).

Key:
- `01` in the beginning of notebook name indicates scripts utilising the analytical object of all immune cells
- `02` in the beginning of notebook name indicates scripts utilising the analytical object of monocytes, macrophages, and cDC2s
- `03` in the beginning of notebook name indicates scripts utilising the analytical object of innate lymphoid cells (ILCs; including NK cells) from the HFRSCA v1 integrated with public datasets from the colon, lung, and oral cavity.
