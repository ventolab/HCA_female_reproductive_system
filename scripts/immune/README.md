# Immune | HFRSCA v1

This directory contains the scripts used for:
1. (`analysis`) Integrating, annontating, and analysing the immune cells in the Human Female Reproductive System Cell Atlas v1 (across lifespan, excluding donors taking exogenous hormones).
2. (`figure_generation`) Generating all immune-related main and supplementary figures.

Key:
- `00` in the beginning of notebook name indicates scripts utilising the analytical object of all immune cells
- `01` in the beginning of notebook name indicates scripts utilising the analytical object of monocytes, macrophages, and cDC2s
- `02` in the beginning of notebook name indicates scripts utilising the analytical object of innate lymphoid cells (ILCs; including NK cells) from the HFRSCA v1 integrated with public datasets from the colon, lung, and oral cavity.
